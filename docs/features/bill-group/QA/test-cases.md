# 账单分组 · 测试用例

## 测试环境
- 设备：iPhone 17 Pro（iOS 26.x）
- 模式：深色 / 浅色
- 主题：Notion / Liquid Glass / Animal Island

| # | 场景 | 前置条件 | 操作步骤 | 预期结果 | 实际结果 | 状态 |
|---|------|---------|---------|---------|---------|------|
| 1 | 新建流水默认分组 | 首次安装 | 打开新建流水 Modal | 「账单分组」显示「日常消费」 | | ⬜ |
| 2 | 新建流水切换分组 | 存在多个分组 | 新建流水 → 选择非默认分组 → 保存 | 列表该行金额下方显示所选分组名 | | ⬜ |
| 3 | AA 账本隐藏分组行 | 已创建 AA 账本 | 新建流水 → 切换到 AA 账本 | 账单分组行隐藏 | | ⬜ |
| 4 | 详情页修改分组 | 存在个人流水 | 详情 → 点击账单分组 → 选新分组 → 保存 | 列表刷新显示新分组名 | | ⬜ |
| 5 | AA 占位流水展示 | AA 分账已结算 | 查看 AA 占位流水 | 显示「AA 分账」不显示分组名 | | ⬜ |
| 6 | 预设分组不可删 | 存在「日常消费」 | 分组选择页左滑「日常消费」 | 无删除按钮 | | ⬜ |
| 7 | 自定义分组可删 | 存在自定义分组 | 分组选择页左滑自定义分组 → 删除 | 分组被软删除 | | ⬜ |
| 8 | Hub 账单分组卡片 | 本月有多分组流水 | 统计 Tab → 滑到账单分组卡 | hero 显示最高分组名+金额 | | ⬜ |
| 9 | 分组详情页 | 本月有流水 | 点击账单分组卡 | 按金额降序排列 | | ⬜ |
| 10 | 本月统计分组区块 | 本月有多分组流水 | 本月统计页 | 分类排行下方 Top5 分组 | | ⬜ |
| 11 | 空分组列表 | 本月无流水 | 账单分组详情页（无数据月） | 空态提示 | | ⬜ |
| 12 | 历史数据迁移 | 有 v7 旧数据 | v7 → v8 升级 | 旧流水 bill_group_id = default-bill-group | | ⬜ |
| 13 | 语音记账默认分组 | 已配置 LLM | 语音录入流水 | 默认归属「日常消费」 | | ⬜ |
| 14 | 三种主题 | 存在流水 | 切换三种主题查看分组相关页面 | 无布局异常 | | ⬜ |
