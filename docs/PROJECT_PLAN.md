## 五、新增需求 / Bug 操作指南

### 新增需求时更新本表：

1. **进行中需求** 表添加一行
2. BA 完成后勾选 BA 列，填入 `docs/features/<name>/` 路径
3. DEV 完成后勾选 DEV 列
4. QA 完成 + 用户验收后移到 **已完成需求** 表

### 新增 Bug 时更新本表：

1. **Bug 修复记录** 表添加一行，填入 `docs/bugfixes/<name>/` 路径
2. DEV 修复完成后更新状态
3. QA 验证通过后标记完成
